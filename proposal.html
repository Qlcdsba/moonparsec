项目概述

moonparsec 是专为 MoonBit 语言原生设计的高性能、零拷贝、类型安全的解析器组合子 (Parser Combinator) 框架。该项目填补了 MoonBit 生态中声明式词法解析框架的空白，允许开发者以构建“乐高积木”的形式，自由组装小型解析单元，轻松应对针对复杂文本协议（如 JSON, CSV, YAML 或自定义 DSL）的高可靠解析。

核心特色与技术优势
零拷贝性能表现：通过指针偏移量 (Offset) 和字符串视图流式匹配，完全避免了解析过程中的中间临时分配，提高 GC 效率。
精准的 Span 追踪：行/列号 (Line/Column) 及物理偏移自动递增，配合 span() 组合子，可精确保存 AST 语法节点的物理起止范围。
编译器级智能报错：独创 ParseError::format，可在出错时提取代码所在行的上下文，输出形如 ^ -- Expected digit 的视觉诊断，利于友好纠错。
无警告纯正适配：完美融入 MoonBit 最新的泛型参数约束、无副作用 try-catch 控制流，在严格的代码检查中保持 100% 编译通过且无任何警告。

坐标解析器编写示例
// 将 "(12, -34)" 解析为 (Int, Int)
fn parse_coordinate() -> Parser[(Int, Int)] {
  let comma = char(',')
  let num = int()
  char('(').preceded(
    num.terminated(comma).and_then(fn(x) {
      num.terminated(char(')')).map(fn(y) {
        (x, y)
      })
    })
  )
}
典型落地应用场景
RFC 4180 兼容的 CSV 解析器：已基于本库实现，稳定应对包含引号转义 "" 以及嵌入逗号/新行的高复杂度表格数据。
支持嵌套递归的 JSON 解析器：支持 Null, Bool, Number, String 以及多级嵌套列表和 Object，树状 AST 输出直观精巧。
未来规划
Wasm 底层融合：适配 Wasm-GC 底层的多语言原生类型映射，使字符串转换效率实现跃升。
标准扩展生态：计划发布对 YAML 及 TOML 语言的支持插件，成为 MoonBit 全场景解析基石。
