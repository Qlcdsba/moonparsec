项目概述

MoonWAT 是专为 MoonBit 语言开发的高性能 WebAssembly 文本格式（WAT）解析与语法诊断分析引擎。由于 MoonBit 深度绑定 WebAssembly 平台，该项目通过底层高度优化的 Parser Combinator 解析核心，构建出针对 WebAssembly 规范中文本格式（S-expression S-表达式）的完整语法树（AST）及编译诊断流，致力于为 MoonBit 开发者提供原生的 Wasm 解析与分析底层工具链。

核心特色与技术优势
完整的 S-Expression 支持：基于 AST 设计，可以完美转换、识别并输出 Wasm 所有的嵌套指令与 S-表达式层级树。
WAT 注释流过滤与追踪：内置自动剥离单行 ;; 以及多行块级 (; ... ;) 注释的解析逻辑，避免对核心 AST 的构建产生噪音。
精确的 Span 与行/列追踪：在解析 Wasm 大文件时自动增量计算行、列，利用 span() 组合子精准标记函数、参数、局部变量等指令集的物理起止范围。
零拷贝与极致报错诊断：继续沿用偏移动态游标流的零拷贝设计，报错时由 ParseError::format 自动提取上下文生成精准的 `^` 诊断信息。

WAT 解析器编写示例
// 递归解析 Wasm 的 S-Expression 层级树
pub fn wat_expr() -> Parser[SExpr] {
  Parser::new(fn(state) {
    let p = wat_list().or(wat_atom())
    p.parse(state)
  })
}

// 过滤并跳过空格与单行/多行注释
pub fn wat_ignored() -> Parser[Unit] {
  let ws = satisfy(fn(c) { c == ' ' ... })
  ws.or(wat_line_comment())
    .or(wat_block_comment())
}
